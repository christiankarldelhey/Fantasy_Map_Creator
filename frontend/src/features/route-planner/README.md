# Route Planner Feature

## Purpose
Plan and visualize routes between locations using different medieval transportation methods.

## Planned Functionality
- Route calculation between multiple waypoints
- Support for different vehicle types (walking, horse, cart, ship)
- Consider terrain and biome types in route planning
- Estimate travel time based on vehicle type and terrain
- Display routes on the map with waypoints

## Status
🚧 Not yet implemented

## Dependencies
- Distance calculator feature
- Biome system feature
- Location and region entities
