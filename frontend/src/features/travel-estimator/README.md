# Travel Estimator Feature

## Purpose
Estimate realistic daily travel distances for characters based on various factors.

## Planned Functionality
- Calculate daily travel distances based on:
  - Transportation method (walking, horse, cart, etc.)
  - Terrain type (flat, hilly, mountainous)
  - Biome type (forest, plains, desert, etc.)
  - Weather conditions
  - Character/party size and composition
- Display travel time estimates on routes
- Compare different travel scenarios

## Status
🚧 Not yet implemented

## Dependencies
- Route planner feature
- Biome system feature
- Elevation system feature
- Climate system feature (for weather)
