# Biome System Feature

## Purpose
Manage and display different biome types (forests, mountains, plains, etc.) on the map.

## Planned Functionality
- Define biome types (forest, mountain, plains, desert, etc.)
- Display biome layers on the map
- Biome-based styling and visualization
- Impact on travel calculations (different speeds through different biomes)
- Biome data management (CRUD operations)

## Status
🚧 Not yet implemented

## Dependencies
- Region entity for biome boundaries
- Map entity for layer management
