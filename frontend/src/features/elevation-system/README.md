# Elevation System Feature

## Purpose
Integrate and display elevation data for the Middle Earth map.

## Planned Functionality
- Display elevation data as a layer
- Elevation-based visualization (hillshading, contour lines)
- Elevation profiles for routes
- Impact on travel calculations (uphill/downhill affects speed)
- 3D terrain visualization (optional)

## Status
🚧 Not yet implemented

## Dependencies
- Map entity for layer management
- Route planner for elevation profiles
- External elevation data source (DEM/GeoTIFF)
